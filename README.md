# Fluffy Desk

療癒系桌面小寵物。麻糬們住在你螢幕上，吃糖、打瞌睡、追游標、互相依偎。

## 特點

- **不打擾工作** — 全透明、無焦點、滑鼠穿透；永遠在最上層但不擋畫面
- **看你動才有事做** — 打字、移動滑鼠會生出碎屑、糖粒、糖塊；長時間不動麻糬會睡著
- **有自己的個性** — AI 狀態機驅動，主動找掉落物、追游標玩、累了回巢窩著
- **物理擬真** — matter.js 模擬掉落、跳躍、堆疊，糖塊滾動手感自然
- **能量量表** — 鍵鼠活動填滿後，從畫面邊滾出大糖塊作為獎勵
- **三種糖味** — 牛奶、抹茶、草莓，視覺與動作各有差別
- **跨平台** — Windows、macOS（Apple Silicon + Intel）
